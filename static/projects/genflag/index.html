Genflag

A local flag generator for CTFs

Feel free to look at the network tab and/or the page source to verify it is local only. The code has been kept barebone for this specific purpose.

Format Specifiers

Specifier Output
{:x} Lowercase hexadecimal
{:X} Uppercase hexadecimal
{:b64} Base64 (with padding)
{:b64-} Base64 (without padding)
{:b64url} Base64url (with padding)
{:b64url-} Base64url (without padding)
{:a} Random characters from a custom alphabet

To include literal braces in the output, wrap the specifier in double braces. For example, FLAG{{:x}} produces FLAG{<hex>}.
